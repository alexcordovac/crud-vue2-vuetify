# CRUD of students management with login

## Description
```
This is a system for student managament, it takes advantage of Vue 2 and Vetify to create a beutiful UI.
It also uses a RestAPI request in order to fetch the data.
```

## Libraries used
```
Vuetify
VueRouter
Vuex
```
